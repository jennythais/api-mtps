API cho gửi mail:
- SendGrid API: (https://sendgrid.com/solutions/email-api/)

// Tạo đối tượng API SendGrid
$sendgrid = new SendGrid(getenv('SENDGRID_API_KEY'));

// Tạo đối tượng email
$email = new SendGrid\Mail\Message();
$email->setFrom('no-reply@example.com');
$email->addTo($to_email);
$email->setSubject('Thông báo tạo tài khoản');
$email->setBody('
Chào [name],
Tài khoản của bạn đã được tạo thành công. Vui lòng nhấp vào liên kết dưới đây để đăng nhập vào hệ thống:
[link]
Liên kết này sẽ hết hạn trong 1 phút. Nếu bạn không thể truy cập liên kết này, vui lòng liên hệ với quản trị viên.
');
// Tạo liên kết có thời hạn
$expiration_time = time() + 60; // 1 phút
$link = "https://example.com/login?name=$name&expiration_time=$expiration_time";

// Thêm liên kết vào email
$email->addLink($link, 'Đăng nhập');

// Gửi email
$sendgrid->send($email);


Account Management:

Create a salesperson account (POST)
Send email for new account creation (automatic)
Login with a one-time link (GET)
Update password on first login (POST)
View user profile (GET)
Update profile picture (POST)
Change password (POST)
Admin-specific APIs (GET):
View staff list
View details of an employee
Resend a 1-minute login email
Lock/unlock an account
View sales information of an employee
Product Catalog Management (for admin only):

View product list (GET)
Add a new product (POST)
Edit product information (PUT)
Delete a product (DELETE)
Customer Management:

Create a customer account (automatic on the first purchase)
View customer's personal information (GET)
View customer purchase history (GET)
View details of a customer's order (GET)
Transaction Processing:

Display the list of selected products (GET)
Add a product to the list by searching by name (POST)
Add a product to the list by entering the barcode (POST)
Display the overview interface with order information (GET)
Complete payment and generate an invoice (POST)
Update cart information instantly (POST)
Reporting and Analytics:

View reports for different timeframes (GET)
View reports for a specific period (GET)
Calculate and display aggregate report information (GET)
Display a list of orders corresponding to the selected timeline (GET)
